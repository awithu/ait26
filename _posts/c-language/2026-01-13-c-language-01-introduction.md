# C언어 강의 1편: C언어 소개 및 개발 환경 설정

## C언어란 무엇일까요?

C언어는 1972년 벨 연구소의 데니스 리치(Dennis Ritchie)가 개발한 프로그래밍 언어입니다. 지금까지도 전 세계에서 가장 널리 사용되는 언어 중 하나예요.

**왜 C언어를 배워야 할까요?**

C언어는 프로그래밍의 기초가 되는 언어입니다. 마치 건물을 지을 때 기초 공사를 하는 것처럼, C언어를 배우면 다른 프로그래밍 언어들도 훨씬 쉽게 이해할 수 있어요. C++, Java, Python 등 많은 언어들이 C언어의 영향을 받았거든요.

## C언어의 특징

### 장점 👍

1. **빠른 실행 속도**
   - C언어는 컴파일 언어라서 실행 속도가 매우 빠릅니다
   - 게임, 운영체제 같은 성능이 중요한 프로그램에 적합해요
   - 마치 자동차로 비유하면, 스포츠카처럼 빠르게 달릴 수 있는 언어입니다

2. **메모리를 직접 제어할 수 있어요**
   - 메모리를 직접 관리할 수 있어서 효율적인 프로그램을 만들 수 있습니다
   - 하지만 이게 단점이 될 수도 있어요 (아래에서 설명할게요)

3. **어디서나 사용 가능**
   - Windows, Mac, Linux, 스마트폰까지 거의 모든 곳에서 사용할 수 있어요
   - 한 번 배우면 다양한 곳에 적용할 수 있습니다

4. **다른 언어의 기초**
   - C언어를 배우면 C++, Java, C# 등 다른 언어를 배우기가 훨씬 쉬워집요

### 단점 👎

1. **메모리 관리가 어려워요**
   - 메모리를 직접 관리해야 해서 실수하기 쉽습니다
   - 초보자에게는 조금 어려울 수 있어요

2. **포인터 개념이 어려워요**
   - 나중에 배우게 될 포인터라는 개념이 처음엔 헷갈릴 수 있습니다
   - 하지만 차근차근 배우면 이해할 수 있어요!

3. **디버깅이 까다로워요**
   - 프로그램에 오류가 생겼을 때 찾아내기가 다른 언어보다 어려울 수 있어요

하지만 걱정하지 마세요! 이 강의를 따라오시면 차근차근 배울 수 있습니다. 😊

## 개발 환경 설정하기

이제 실제로 C언어로 프로그램을 작성할 수 있도록 개발 환경을 설정해볼게요.

### Windows 환경 설정

#### 1단계: MinGW-w64 설치하기

MinGW-w64는 Windows에서 C언어를 컴파일할 수 있게 해주는 도구예요. 컴파일이 뭔지 모르셔도 괜찮아요, 나중에 설명드릴게요!

**설치 방법:**
1. https://www.mingw-w64.org/ 접속
2. Downloads 메뉴에서 Windows용 설치 파일 다운로드
3. 설치 프로그램 실행 후 "Next"를 계속 클릭
4. 설치가 완료되면 컴퓨터를 재시작해주세요

**설치 확인:**
명령 프롬프트(CMD)를 열고 다음 명령어를 입력해보세요:
```bash
gcc --version
```
버전 정보가 나오면 성공입니다! 🎉

#### 2단계: Visual Studio Code 설치하기

코드를 작성할 에디터가 필요해요. Visual Studio Code(줄여서 VS Code)를 추천합니다. 무료이고 사용하기 쉬워요!

**설치 방법:**
1. https://code.visualstudio.com/ 접속
2. "Download for Windows" 클릭
3. 다운로드한 파일 실행하여 설치
4. 설치 후 VS Code 실행

**C/C++ 확장 프로그램 설치:**
1. VS Code 실행
2. 왼쪽 사이드바에서 확장 프로그램 아이콘 클릭 (또는 `Ctrl + Shift + X`)
3. 검색창에 "C/C++" 입력
4. Microsoft의 "C/C++" 확장 프로그램 설치

#### 3단계: 첫 번째 프로그램 작성하기

이제 드디어 첫 번째 C언어 프로그램을 작성해볼게요!

**파일 만들기:**
1. VS Code에서 새 파일 만들기 (`Ctrl + N`)
2. `hello.c`로 저장 (`Ctrl + S`)
3. 다음 코드를 입력하세요:

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    printf("안녕하세요! C언어를 배우고 있어요!\n");
    return 0;
}
```

**코드 설명:**
- `#include <stdio.h>`: 화면에 글자를 출력하는 기능을 사용하기 위해 필요한 부분이에요. 지금은 "필수 마법 주문"이라고 생각하시면 돼요!
- `int main() { }`: 모든 C언어 프로그램은 여기서 시작해요. 프로그램의 시작점입니다.
- `printf("...")`: 화면에 글자를 출력하는 함수예요. 괄호 안의 내용이 화면에 나타납니다.
- `\n`: 줄바꿈을 의미해요. Enter 키를 누른 것과 같습니다.
- `return 0;`: 프로그램이 정상적으로 끝났다는 의미예요.

#### 4단계: 프로그램 실행하기

**방법 1: VS Code 터미널 사용 (추천)**
1. VS Code에서 `Ctrl + `` (백틱)을 눌러 터미널 열기
2. 다음 명령어 입력:
```bash
gcc hello.c -o hello
```
이 명령어는 `hello.c` 파일을 컴파일(번역)해서 `hello.exe`라는 실행 파일을 만드는 거예요.

3. 실행 파일 실행:
```bash
hello
```
또는
```bash
./hello
```

**방법 2: 명령 프롬프트 사용**
1. `hello.c` 파일이 있는 폴더로 이동
2. 위와 동일한 명령어 입력

**출력 결과:**
```
Hello, World!
안녕하세요! C언어를 배우고 있어요!
```

와! 첫 번째 프로그램이 실행되었어요! 🎉

## 프로그램 구조 이해하기

모든 C언어 프로그램은 비슷한 구조를 가지고 있어요. 자세히 살펴볼게요:

```c
#include <stdio.h>  // 1. 필요한 라이브러리 불러오기

int main() {        // 2. 프로그램 시작점
    // 여기에 코드를 작성합니다
    printf("안녕하세요!\n");
    return 0;       // 3. 프로그램 종료
}
```

### 각 부분 설명

1. **`#include <stdio.h>`**
   - 표준 입출력(Standard Input/Output) 라이브러리를 불러오는 거예요
   - `printf` 같은 함수를 사용하려면 반드시 필요합니다
   - 지금은 "필수 준비물"이라고 외우시면 돼요!

2. **`int main() { }`**
   - 모든 C언어 프로그램은 `main` 함수에서 시작해요
   - `int`는 정수를 의미하는데, 나중에 배울 거예요
   - 프로그램이 실행되면 여기부터 차례로 실행됩니다

3. **`return 0;`**
   - 프로그램이 정상적으로 끝났다는 의미예요
   - 0은 "에러가 없다"는 뜻입니다
   - 다른 숫자를 반환하면 에러가 발생했다는 의미예요

## 실전 예제: 자기소개 프로그램

이제 배운 내용으로 간단한 자기소개 프로그램을 만들어볼게요:

```c
#include <stdio.h>

int main() {
    printf("=== 자기소개 ===\n");
    printf("이름: 홍길동\n");
    printf("나이: 25세\n");
    printf("취미: 프로그래밍\n");
    printf("좋아하는 언어: C언어\n");
    printf("================\n");
    
    return 0;
}
```

**출력 결과:**
```
=== 자기소개 ===
이름: 홍길동
나이: 25세
취미: 프로그래밍
좋아하는 언어: C언어
================
```

여러분도 자신의 정보로 바꿔서 실행해보세요!

## 컴파일이란?

처음에 "컴파일"이라는 말을 들었을 때 헷갈리셨을 수도 있어요. 간단히 설명드릴게요:

**컴파일 = 번역**

- 우리가 작성한 C언어 코드는 사람이 읽기 쉬운 형태예요
- 하지만 컴퓨터는 이걸 직접 이해하지 못해요
- 그래서 컴파일러가 우리 코드를 컴퓨터가 이해할 수 있는 기계어로 번역해주는 거예요!

**비유하면:**
- 우리가 한국어로 글을 쓰면 (C언어 코드)
- 번역기가 영어로 번역해주는 것 (컴파일)
- 영어를 읽을 수 있는 사람이 읽는 것 (컴퓨터가 실행)

## 자주 하는 실수와 해결 방법

### 실수 1: 세미콜론(;) 빼먹기
```c
printf("Hello")  // ❌ 에러!
printf("Hello"); // ✅ 올바름
```
C언어에서는 문장 끝에 세미콜론을 반드시 붙여야 해요!

### 실수 2: 따옴표 사용 잘못하기
```c
printf('Hello'); // ❌ 작은따옴표는 문자 하나만 가능
printf("Hello"); // ✅ 큰따옴표 사용
```

### 실수 3: 한글 인코딩 문제
한글이 깨져서 나올 수 있어요. VS Code에서 파일을 저장할 때 인코딩을 UTF-8로 설정해주세요.
